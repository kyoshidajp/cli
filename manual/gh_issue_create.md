## gh issue create

Create a new issue

### Synopsis

Create a new issue

```
gh issue create [flags]
```

### Options

```
  -b, --body string    Supply a body. Will prompt for one otherwise.
  -t, --title string   Supply a title. Will prompt for one otherwise.
  -w, --web            Open the browser to create an issue
```
